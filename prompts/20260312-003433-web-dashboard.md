## Key decisions

**This prompt log covers the web dashboard** (PR #112).

**Inline HTML, no build step**: The entire dashboard is a single HTML string embedded
in Python. No webpack, no React, no static file serving. This keeps the container
simple (no static asset directory) and the dashboard loads in one request. Total
size is under 30KB.

**8 panels over the existing dashboard**: The project already had a dashboard module
(`src/robotocore/dashboard/app.py`). The agent extended it with State/Snapshots and
Health panels, plus a thin `handler.py` re-export module for cleaner imports.

**Dark theme**: Developer tools should be dark. Light text on dark background,
monospace font for data cells, color-coded status dots (green/yellow/red) for
health. No external fonts or CDN resources — works fully offline.

**Vanilla JS with fetch()**: Each panel fetches its data from the existing
`/_robotocore/*` API endpoints. No framework overhead. Auto-refresh is a 5-second
`setInterval` toggle — simple and effective for a dev tool.
